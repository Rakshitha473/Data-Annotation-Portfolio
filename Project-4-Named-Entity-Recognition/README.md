# Named Entity Recognition (NER) Annotation Project

## Project Overview

This project demonstrates a complete Named Entity Recognition (NER) annotation workflow using Label Studio.

The objective of this project was to identify and annotate named entities within text using predefined labels. The dataset contains realistic sentences from business, travel, education, banking, healthcare, and general communication scenarios.

---

## Project Objective

Annotate named entities in text using the following labels:

- Person
- Organization
- Location
- Date
- Time
- Money
- Email
- Phone Number

---

## Dataset Information

- Total Sentences: 50
- Dataset Format: JSON
- Annotation Tool: Label Studio 1.23.0

The dataset includes examples related to:

- Business
- Education
- Banking
- Healthcare
- Shopping
- Travel
- Technology
- Customer Support

---

## Annotation Workflow

1. Created a new Named Entity Recognition project in Label Studio.
2. Configured the labeling interface using XML.
3. Imported the JSON dataset.
4. Annotated all named entities by highlighting text spans.
5. Reviewed annotations for consistency.
6. Exported the completed annotation file.

## Challenges

Some sentences contained multiple entities of different types. Careful attention was required to correctly identify entity boundaries and assign the appropriate label while maintaining consistency throughout the project.

Examples included:

- Multiple entities in a single sentence
- Landmark versus location identification
- Dates versus relative time expressions
- Organization names appearing with person names

---

## Outcome

Successfully completed annotation of 50 sentences containing multiple entity types.

The project demonstrates practical experience with Named Entity Recognition (NER) workflows commonly used in Natural Language Processing (NLP) and AI model training.
