# Intent Classification Annotation Project

## Objective
Annotate customer queries by identifying the user's primary intent using predefined intent categories.

## Labels Used

### Intent Categories
- Book Flight
- Cancel Booking
- Track Order
- Return Product
- Account Support
- Payment Issue
- Technical Support
- General Inquiry

## Dataset
50 customer queries covering:
- Flight booking requests
- Booking cancellations
- Order tracking
- Product returns
- Account-related issues
- Payment problems
- Technical issues
- General customer inquiries

## Tool Used
- Label Studio 1.23.0

## Skills Demonstrated
- Intent Classification
- Text Classification
- Natural Language Processing (NLP)
- Data Annotation
- Label Studio
- Customer Support Data Annotation
- Dataset Review
