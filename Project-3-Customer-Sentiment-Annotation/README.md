# Customer Sentiment Analysis

## Project Overview
This project demonstrates a complete text annotation workflow using Label Studio.

The objective was to classify customer reviews based on their overall sentiment and identify the primary aspect discussed in each review.

---

## Objective

Annotate customer reviews by assigning:

### Sentiment
- Positive
- Neutral
- Negative

### Category
- Product Quality
- Delivery
- Customer Service
- Pricing
- Packaging
- Other

---

## Dataset

- Total Reviews: 60
- Format: JSON
- Tool Used: Label Studio 1.23.0

The dataset contains customer feedback covering:
- Product quality
- Delivery experience
- Customer service
- Packaging
- Pricing
- Returns and refunds

---

## Annotation Workflow

1. Created a Label Studio project.
2. Configured the labeling interface using XML.
3. Imported the JSON dataset.
4. Annotated all 60 customer reviews.
5. Reviewed annotations for consistency.
6. Exported the completed annotations.

---

## Skills Demonstrated

- Text Classification
- Sentiment Analysis
- Multi-label Annotation
- Label Studio
- JSON Data Handling
- Annotation Quality Review

---

## Challenges

Some reviews expressed mixed opinions (for example, positive product quality but negative delivery). These required careful reading to assign the most appropriate sentiment and category while maintaining consistency.

---

## Outcome

Successfully completed annotation of all 60 customer reviews and exported the final annotation dataset.
