# Toxic Comment Detection Annotation Project

## Objective
Annotate online comments by identifying whether they contain toxic or non-toxic language.

## Labels Used

### Toxicity
- Toxic
- Non-Toxic

## Dataset
50 online comments covering:
- Personal insults
- Offensive language
- Abusive comments
- Constructive criticism
- Neutral statements
- Positive feedback
- Customer complaints
- Respectful conversations

## Tool Used
- Label Studio 1.23.0

## Skills Demonstrated
- Toxic Comment Detection
- Content Moderation
- Text Classification
- Data Annotation
- Label Studio
- AI Safety Annotation
- Dataset Review
